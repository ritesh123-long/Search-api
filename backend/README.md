# YouTube Music Search (no-key prototype)

Simple Express API using `yt-search` that does not require any Google API key.

## Features
- YouTube Music URL included
- High-quality thumbnails (maxresdefault)
- Fast search using yt-search
- NodeCache caching
- Rate-limited API
- Ready for Render deploy

## Example Request
https://your-app.onrender.com/search?q=arijit+singh&maxResults=5
